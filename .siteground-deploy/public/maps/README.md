# Carpeta para Planos del Proyecto

Esta carpeta contiene las imágenes de los planos para el Sistema Comercial.

## Formatos soportados
- **SVG** (recomendado) - Mejor calidad para zoom
- **PNG** - Imágenes de alta resolución
- **JPG** - Fotografías o renders

## Estructura de nombres
Los archivos se guardan con el formato:
```
{nombre-proyecto}-{timestamp}.{extension}
```

Ejemplo: `residencial-los-jardines-1707451234567.svg`

## Cómo agregar un plano manualmente
1. Copia tu archivo SVG o PNG a esta carpeta
2. Renómbralo con un nombre descriptivo (ej: `mi-proyecto.svg`)
3. Configura la URL del plano en el proyecto desde el panel de administración
